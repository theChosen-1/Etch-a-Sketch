# Etch-a-Sketch

A browser-based Etch-a-Sketch drawing application built with vanilla JavaScript, HTML, and CSS.

## Live Demo

[View Live Demo](https://thechosen-1.github.io/Etch-a-Sketch/)

## About

This project is part of my learning journey through [The Odin Project](https://www.theodinproject.com/) curriculum. It demonstrates DOM manipulation, event handling, and dynamic element creation using JavaScript.

## Features

- Interactive drawing grid that responds to mouse hover
- Customizable grid size (1-100 squares per side)
- Clean, responsive layout
- Dynamic grid generation without page reload

## How to Use

1. Hover your mouse over the grid squares to draw
2. Click the "Resize Grid!" button to change the grid size
3. Enter a number between 1 and 100 for the new grid dimensions

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript

## What I Learned

- Creating and appending DOM elements dynamically
- Event listeners and mouse events
- CSS Flexbox for grid layouts
- Input validation and user prompts
- Calculating element sizes based on container dimensions
